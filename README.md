# Emergency Ping

A full-stack application with a Node.js backend and a React frontend styled with Tailwind CSS. This project is designed to demonstrate the integration of a RESTful API with a modern frontend framework.

## Features

- Node.js backend serving a simple API
- React frontend with Tailwind CSS for styling
- CORS enabled for cross-origin requests

## Technologies Used

- **Backend**: Node.js, Express
- **Frontend**: React, Tailwind CSS
- **Others**: npm, CORS

## Getting Started

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v14 or higher)
- npm (comes with Node.js)

## Run local backend

    Set up the backend

    ```Shell
    cd backend
    ```

    Install the backend dependencies:

    ```Shell
    npm install
    ```

    Run the backend:

    ```Shell
    cd backend
    node server.js
    ```

## Run local frontend

    Set up the frontend:

    ```Shell
    cd /web
    npm install
    ```
    Run the frontend:

    ```Shell
     cd web
    npm start
    ```
